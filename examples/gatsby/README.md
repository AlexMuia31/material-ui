# Gatsby example

## How to use

Download the example [or clone the repo](https://github.com/mui-org/material-ui):

```bash
curl https://codeload.github.com/mui/material-ui/tar.gz/v1.x | tar -xz --strip=2  material-ui-1.x/examples/gatsby
cd gatsby
```

Install it and run:

```bash
npm install
npm run develop
```

## The idea behind the example

[Gatsby](https://github.com/gatsbyjs/gatsby) is a static site generator for React.
